# Sentes Solar Website

Sentes Solar Enerji A.Ş. resmi web sitesi.

## Proje Yapısı

- `index.html` - Ana sayfa
- `about.html` - Hakkımızda sayfası
- `projects.html` - Projeler sayfası
- `stations.html` - Santraller sayfası
- `contact.html` - İletişim sayfası
- `css/` - Stil dosyaları
- `js/` - JavaScript dosyaları
- `assets/` - Görseller ve videolar

## Teknolojiler

- HTML5
- CSS3
- JavaScript

## Kurulum

Projeyi klonlayın ve tarayıcıda `index.html` dosyasını açın.

```bash
git clone https://github.com/OsmanEmreKaya/sentessolar_website.git
cd sentessolar_website
```
